# Trade Risk Manager

A responsive, client-side web application designed to help active traders calculate precise position sizing based on absolute risk parameters. 

## Features
- Calculates absolute capital at risk based on account balance and risk percentage.
- Determines the exact distance to the stop-loss invalidation level.
- Calculates maximum position size across various asset classes by utilizing a user-defined point/pip value.
- Responsive, modern UI built without external libraries.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Custom Styling)
- Vanilla JavaScript (DOM Manipulation, Event Listeners)

## How to Use
1. Enter your total account balance.
2. Enter the maximum percentage you wish to risk.
3. Enter your intended entry and stop loss prices.
4. Enter the dollar value of 1 point/pip for a standard unit of your chosen asset.
5. Click "Calculate Position Size" to view your strict risk parameters.
